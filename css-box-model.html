    • border-radius

    border-radius에 관한 MDN 문서

    border-radius는 px이나 %로 표현한다(가능한 %를 사용). 만일 50%로 설정한다면, 박스의 모양은 원형이 된다

    border-radius를 이용해 모서리를 둥글게 만들 때, box 자체의 모양이 정사각이 아닌 직사각일 경우 %를 사용하면 지금과 같이 못생기게 나온다. 이 경우 px 단위를 사용하면 문제가 해결된다

    border-radius는 img element 등에도 적용 되므로, 사진, 그림의 테두리 모양을 둥글게 하는데에도 사용할 수 있다!
